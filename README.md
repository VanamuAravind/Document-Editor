# **HEY THANKS FOR VISITING MY REPO**
## **THIS IS A NODE BASED DOCUMENT EDITING WEBSITE WITH THE FOLLOWING**
### * **user authentication and authorization**
### * **saved documents for every user**
### * **a good editing canvas with dynamic resizability**
### * **can add multiple number of nodes of different types ( textfields, images, tables, videos, drawing canvas, links, etc. )**
### * **detailed view of every node**
### * **each node is customizable**
